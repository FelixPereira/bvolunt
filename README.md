Sistema de gestão de Voluntários e Organizações Sociais

Este é um projecto de sistema de gerenciamento de voluntários, desenvolvido para facilitar o cadastro e gerenciamento de voluntários, organizações sociais e eventos relacionados.

Mais detalhes em breve.


Recursos principais
Cadastro de voluntários: Os voluntários podem se cadastrar no sistema, fornecendo informações pessoais, habilidades, interesses e disponibilidade.

Cadastro de organizações sociais: As organizações sociais podem se cadastrar no sistema, fornecendo informações sobre a missão, atividades e necessidades da organização.

Cadastro de eventos: Os eventos relacionados a atividades sociais podem ser cadastrados, com informações sobre a data, local, descrição e requisitos de voluntários.

Associação de voluntários a organizações: Os voluntários podem se associar a organizações sociais, demonstrando interesse em apoiar seus projetos e atividades.

Participação em projetos sociais: Os voluntários podem se cadastrar para participar de projetos sociais específicos, demonstrando interesse em contribuir para uma causa específica.

Tecnologias utilizadas
Front-end: React.js com componentes reutilizáveis e estilizados usando a biblioteca Tailwind CSS.

Back-end: Node.js com o framework Express para a criação de APIs RESTful.

Banco de dados: Utilizamos o PostgreSQL para armazenar os dados relacionais do sistema.

Autenticação e autorização: NextAuth.js para autenticação de usuários e controle de acesso aos recursos.
